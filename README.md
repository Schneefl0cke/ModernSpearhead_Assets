# ModernSpearhead_Assets
Assets made for modern spearhead

Ranges on the Tokens are in kilometers. Either written as 1,6km or 800 meter.
Some very modern tokens are made by me, with made up values that hopefully correspond to the others.

Not all pictures are correct, but I work with what I have. They will be updated in time.

The tokens are made to print them on colored paper or to use them in the tabletop simulator where they can be easily imported and colored.
